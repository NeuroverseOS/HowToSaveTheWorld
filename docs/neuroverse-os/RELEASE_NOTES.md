# NeuroVerse OS — Release Notes

This document tracks version history and changes for NeuroVerse OS.

---

## Version 1.0.0 — Initial Release

**Release Date:** January 2025

### 🎉 First Public Release

NeuroVerse OS v1.0.0 is the first release of the world's first intercognitive operating system. This version includes the complete Eight-Box Scaffold, Echelon Core Intelligence, and all foundational systems for building human-AI reasoning partnerships.

---

### ✨ Core Features

#### Eight-Box Scaffold
- Deterministic prompt assembly engine
- Box-Stage Map for auditable box activation
- Support for 7 mission stages (BRIEFING, DRILL1, VIDEO, HP, DRILL2, DEBRIEF, FINAL)
- Modular box architecture (Box 1-8)

#### Echelon Core
- Mythic-tech personality with ICF coaching enforcement
- Mode-aware cognition (Training, Design, Build, Lead)
- Archetype-calibrated coaching style
- First-person immersive narrative voice
- Callsign-based operator addressing

#### Operator Model
- UUID + Callsign identity system
- 9 Canonical Archetype System (Primary, Shadow, Rising)
- Trait, Shadow, and Power tracking
- Short-term memory (last 10 decisions)
- Long-term memory (philosophy statements)

#### Mission Engine
- 7-stage lesson orchestration
- Automatic content delivery on stage advancement
- 5-point reflection architecture (Standard, Micro-Insight, Exercise, Deep)
- Field Guide compilation from reflections

#### State Engine
- Local-first state management (localStorage + IndexedDB)
- Optional Supabase cloud sync
- State versioning and migration support
- Encryption for sensitive operator data

#### ACE Loader
- Local-first knowledge retrieval architecture
- Keyword-triggered content matching
- ACE Box 08 (System Literacy) included
- Extensible plugin system for future ACE Boxes

#### Mode Engine
- Multi-mode cognition framework
- Training Mode (coaching and lesson delivery)
- Design Mode (strategic thinking and systems design)
- Build Mode (execution planning and task breakdown)
- Lead Mode (people leadership and influence)

#### Archetype System
- 12-scenario assessment engine
- Primary/Shadow/Rising calculation algorithms
- Trait inference from reflection patterns
- 9 Canonical Archetypes:
  - WATCHTOWER (The Observer)
  - WEAVER (The Connector)
  - VEIL (The Guardian)
  - OPERATOR (The Executor)
  - ENGINE (The Builder)
  - LUMEN (The Illuminator)
  - CIPHER (The Analyst)
  - DRIFT (The Adapter)
  - CHRONICLE (The Archivist)

#### Identity System
- Vanguard callsign generation (Greek letter + number)
- UUID-based unique operator identification
- Activation ceremony architecture
- Bonding protocol (Foxhole Protocol integration)

#### Speech Layer
- Text-to-speech integration (OpenAI TTS)
- Browser TTS fallback for offline mode
- Voice model selection (onyx, alloy, echo, nova, shimmer, fable)
- Speed, pitch, volume controls
- Critical principle: Voice is cosmetic, identity is sovereign
