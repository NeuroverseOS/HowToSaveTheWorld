## 📨 How to Submit a Support Request

### Email Template:

**Subject Line Format:**
```
[Category] Brief description
```

Examples:
- `[Licensing] Request for Enterprise License`
- `[Technical] Installation error on Node v20`
- `[Bug Report] Echelon chat timeout in Design Mode`

**Email Body:**

```
Name: [Your Name]
Organization: [Your Company/Project]
License Type: [Startup / Enterprise / Web3 / None]
License Key: [Your Key] (if applicable)

Category:
[ ] Licensing
[ ] Technical Support
[ ] Architectural Question
[ ] Bug Report
[ ] Security Issue
[ ] Other

Description:
[Detailed description of your question or issue]

Expected Behavior: (if applicable)
[What you expected to happen]

Actual Behavior: (if applicable)
[What actually happened]

Steps to Reproduce: (if applicable)
1. [First step]
2. [Second step]
3. [...]

Environment: (if applicable)
- NeuroVerse OS version: [e.g., 1.0.0]
- Node.js version: [e.g., 18.16.0]
- Operating System: [e.g., macOS 13.4]
- AI Provider: [e.g., OpenAI GPT-4]

Additional Context:
[Any other relevant information]
```

## ⏰ Response Times

We aim to respond to all inquiries within **48 hours**.

### Priority Levels:

| Priority | Response Time | Description |
|----------|---------------|-------------|
| **Critical** | 24 hours | Security vulnerabilities, production outages |
| **High** | 48 hours | Technical bugs, license issues |
| **Medium** | 3-5 days | Feature requests, architectural questions |
| **Low** | 1 week | General inquiries, documentation questions |

---

## 🚫 What We Cannot Help With

- **Unlicensed usage** — You must have a valid license to receive technical support
- **Custom development** — We do not provide custom development services for licensees' applications
- **Reverse engineering assistance** — We do not help with extracting or cloning NeuroVerse OS internals
- **Third-party integrations** — We do not provide support for third-party tools or libraries
- **General AI/LLM questions** — Support is limited to NeuroVerse OS-specific issues
