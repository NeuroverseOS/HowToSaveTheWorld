# NeuroVerse Canon — World Rules & Protocol Definitions

This repository contains the canonical rules, protocols, and world architecture of the NeuroVerse.

These canonical documents describe world rules. They are referenced by applications (such as How to Save the World) but are not defined or controlled by those applications.

---

## 📜 What Is the Canon?

The **NeuroVerse Canon** defines the fundamental rules, protocols, and rituals that govern the NeuroVerse world system.

These are:
- **World rules** — Not application features
- **Protocol definitions** — Not implementation code
- **Canonical rituals** — Not UI flows
- **Fundamental laws** — Not configurable settings

Think of the Canon as the "physics" of the NeuroVerse. Applications like **How to Save the World** operate *within* these rules but do not define them.
